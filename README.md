# Lankalapalli Guna - Personal Portfolio

A sleek, modern, and minimalist personal portfolio website built with vanilla web technologies. It showcases my skills in Data Structures and Algorithms (DSA), backend engineering, and my strong competitive programming background.

## 🛠️ Built With

*   **HTML5:** Semantic and accessible structure.
*   **CSS3:** Custom styling, CSS Grid/Flexbox layouts, responsive design, and smooth animations without any external UI frameworks.
*   **Vanilla JavaScript:** Minimal scripting for interactive elements and smooth scrolling.

## ✨ Features

*   **Responsive Design:** Mobile-first approach ensuring a seamless experience across all devices.
*   **Modern Aesthetics:** Clean typography, balanced visual hierarchy, deep dark theme, and subtle micro-interactions/animations.
*   **Competitive Programming Spotlight:** A dedicated, visually dominant section highlighting achievements across platforms like LeetCode, CodeChef, and Codeforces.
*   **Projects Showcase:** Card-based layouts detailing backend-heavy projects (Grocery Delivery App, QuickChat Messaging Platform) with their respective tech stacks.
*   **Performance Optimized:** Fast loading times, native browser features, and clean code without bloat.

## 📁 Project Structure

```text
.
├── assets/             # Images, icons, and other static assets (profile, projects, CP stats)
├── index.html          # Main HTML document
├── script.js           # Interactive functionality (e.g., image modals)
└── style.css           # Global styles and responsive layouts
```

## 💻 Running Locally

To view the project locally, simply clone the repository and open the `index.html` file in your preferred web browser. No build steps, bundlers, or package managers required!

```bash
# Clone the repository
git clone https://github.com/guna2007/portfolio.git

# Navigate to the project directory
cd portfolio

# Open index.html in your browser
```
